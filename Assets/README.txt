Controls:

TOP DOWN VIEW:
- Buttons: select object (hover over playing field to see where object will be placed
- Key "E": rotate object by 90 degrees
- Left Mouse Button: place objects on playingfield 
	(FriendAI will spawn automatically at spawn point when game mode is switched)
- Enter: switch game mode

FIRST PERSON VIEW:
- Keys "W A S D": Move
- Mouse: look around
- Space: jump
- Left Shift: run
- Left Mouse Button: shoot weapon
- Enter: switch game mode

Win: enemy base has zero health
Loss: player base has zero health OR player dies

You can switch as much as you like between the two game modes (to buy new defences etc.) but every time you do the enemy can buy new minions
